
# CreativeFlow Content Hub

A professional dashboard for managing social media content, production status, and creative assets. Built as an educational project for portfolio purposes.

## Key Features
- **AI Content Optimization**: Powered by Google Gemini AI.
- **Attendance System**: GPS-confirmed digital presence for ambassadors.
- **Social Media Analytics**: Data-driven performance tracking.
- **Collaborative Scheduling**: Multi-ambassador task assignments.

## Technology Stack
- **Frontend**: React (ESM) + Tailwind CSS
- **AI Integration**: Google Generative AI (Gemini 3 Flash)
- **Deployment**: Optimized for GitHub Pages

## Disclaimer
This is an **Educational Project - Powered by Gemini AI**. It is for demonstration purposes only and does not collect sensitive user data.
